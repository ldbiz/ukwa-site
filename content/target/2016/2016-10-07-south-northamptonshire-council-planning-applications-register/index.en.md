The online planning application register is provided for users to search for planning applications submitted since 1974. It is also possible to search by many other fields including Decision Dates, Appeals Dates (for applications received after 1988 only), Agents and Applicants Names (for applications received after 1988 only), Application Category (for applications received after 1993 only) and other useful fields.
